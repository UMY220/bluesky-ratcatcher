Bluesky Moderation List Generator

(Silence, cockroach)

Instructions

  1. Copy the URL of a user into the User URL box
  2. Click Get ID button
  3. Click View Page to review the user's page
  4. Click Followers to load a list of the user's followers
  5. Click Mark Reviewed to mark the page as "good" so you don't have to review it again
  6. Click Add To List to add the user to the selected list
